# JamContest Monetization Model
**📅 Güncellenme: 19 Mayıs 2026 | Türkiye gerçek rakamları**

**Tanım:** Yaratıcı ekosistem platformu. Sosyal feed + portföy + adil yarışmalar + işe alım.
Misyon: Yaratıcıları sömürmek değil, onların emeğini ve görünürlüğünü değerlendir.

---

## 💰 Kullanıcı Tipleri ve Para Kazanma

### 1️⃣ **SANATÇI / YARATICI** (Creators)

#### Davranış
- Portföy oluşturur, sosyal feedde paylaşır
- Günde 1-3 post (tier'e bağlı)
- Yarışmalara katılır, ödül kazanır
- Diğer yaratıcılardan iş teklifi alır

#### Para Kazanma Kanalları

1. **Yarışma Ödülleri** ✅
   - 1. ye 7.500 TL, 2. ye 4.000 TL, 3. ye 2.000 TL
   - Veya kategoriye göre: 5.000 - 25.000 TL
   - Sponsor/bağışçılardan gelen para dağılır
   - Platform 0% komisyon (sanatçı %100 alır)

2. **İş/Freelance Teklifler** ✅
   - Şirket/bireyden gelen "Seni işe alabilir miyiz?" teklifi
   - Sanatçı kabul/reddet
   - Ücret: Taraflar arası direkt anlaşma, platform aracı değil
   - Freelance saatlik referansı: 150-400 TL

3. **Premium Features (Tier Sistemi)** ✅

| Tier | Aylık Ücret | Günlük Post | Proje İlanı | İş İlanı | Özellikler |
|------|-------------|-------------|-------------|----------|-----------|
| **Free** | 0 TL | 1 | Hayır | Hayır | Standart profil |
| **Creator** | 59 TL | 5 | Evet (10/ay) | Evet (5/ay) | Creator Badge |
| **Pro** | 149 TL | Sınırsız | Evet (50/ay) | Evet (Sınırsız) | Pro Badge + Featured |

   **Neden bu fiyatlar?**
   - Creator (59 TL): Yazılımcı maaşının %0.09'i = makul
   - Pro (149 TL): Yazılımcı maaşının %0.22'si = Premium yeterli

4. **Bağışçı/Patreon Benzeri Desteği** ✅
   - Platformda "Bana destek ol" butonu
   - Aylık tiers: 25 TL / 75 TL / 250 TL
   - Bağışçı, platform tarafından katlanır (kesinti yok)
   - Sanatçının portföyünde "Supported by" badge

#### Örnek Gelir Harita (Ay başında)

```
Sanatçı A (Free tier):
- Yarışma ödülü: 5.000 TL (iyi sırada tutturdu)
- Freelance iş: Kendi buldu (platform aracı değil)

Sanatçı B (Creator tier @ 59 TL):
- Aylık tier: -59 TL
- Yarışma ödülü: 2.000 TL (3. sıra)
- 3 freelance iş (100 TL/sa × 20 sa = 6.000 TL)
= Net: +7.941 TL

Sanatçı C (Pro tier @ 149 TL):
- Aylık tier: -149 TL
- Yarışma ödülü: 7.500 TL (1. sıra)
- 5 iş ilanı açıp 2'si sonuç verdi (direkt bağlantı)
- 2 Patreon supporter @ 75 TL = 150 TL
= Net: +8.501 TL
```

### 2️⃣ **ORGANIZATÖR / YARIŞMA KURUCUSU** (Contest Creator)

#### Davranış
- Yarışma oluşturur, jüri davet eder, katılımcıları yönetir
- Ödül havuzu belirler, sonuç yayınlar

#### Para Kazanma Kanalları

1. **Yarışma Sponsor Paketi** ✅
   - Şirket/bağışçı: "Biz 50.000 TL ödül koymak istiyoruz"
   - Platform sağladığı: Jüri seçimi, yasal yönetim, şeffaf puanlama
   - Platform ücret: **7% yönetim ücreti = 3.500 TL**
   - (Sanatçılar 50.000 TL'nin tamamını alır)

2. **Kurumsal Yarışma Paketleri** ✅
   - Kapalı yarışma (şirket içi, davetli)
   - Custom jüri seçimi + mentoring
   - Raporlama ve analiz
   - **Paket: 7.500 TL / 15.000 TL / 35.000 TL**

3. **Sponsorluk Ek Paketleri** ✅
   - Logo visibility, sosyal medya mention
   - **Paket: 1.000 TL / 2.500 TL / 5.000 TL**

#### Örnek Flow

```
Ege Üniversitesi: "Design Hackathon 50.000 TL ödül"
→ JamContest: "Jüri, yasal, platform yönetimi için 3.500 TL (7%)"
→ Sanatçılar: 50.000 TL dağılır (kesinlikle %100)
→ Patner sponsor + platform fee = 5.000 TL extra
= JamContest gelir: 8.500 TL
```

---

### 3️⃣ **JÜRİ ÜYESI** (Evaluator)

#### Davranış
- Yarışmalara davet edilir, puanlar verir
- Puanlaması sanatçıya görünür, şeffaf

#### Para Kazanma Kanalları

1. **Puanlama Ücretleri** ✅
   - Saatlik ücret: 200-250 TL (freelance referansı)
   - Katılımcı sayısına göre:
     - 10-50 katılımcı: **500 TL** (~3 saat)
     - 50-150 katılımcı: **1.200 TL** (~6 saat)
     - 150+: **2.500 TL+** (10+ saat)
   - Para nereden? Sponsor paketi / platform geliri

2. **Workshop / Mastermind Oturumları** (ileride) ✅
   - Live session, feedback, mentoring
   - Sanatçılar: Başına 49 TL
   - Jüri: 60-70% payı = 30-35 TL × 20 kişi = 600-700 TL/session

#### Örnek Flow

```
Design yarışması: 120 katılımcı, 5 jüri
Yarışma sponsor paketi: 50.000 TL ödül + 7% = 3.500 TL fee

Jüri A: 1.200 TL alır (6 saatlik iş)
Jüri B-E: 1.200 TL alır (4 jüri × 1.200 = 4.800 TL)
= Toplam jüri: 6.000 TL
= Platform net: 3.500 TL - 6.000 TL = -2.500 TL (sponsor paketten çıkar)
```

### 4️⃣ **ŞİRKET / RECRUITER** (B2B)

#### Davranış
- Profil oluşturur, sanatçı arar, iş teklifler gönderir
- Sponsorlu yarışmalar açar

#### Para Kazanma Kanalları

1. **Recruiter Paketleri (Aylık)** ✅

| Paket | Aylık Ücret | Profil View | İş İlanı | Featured | 
|-------|-------------|-------------|----------|----------|
| **Basic** | 349 TL | 200/ay | 5/ay | Hayır |
| **Pro** | 799 TL | Sınırsız | 20/ay | Evet |
| **Enterprise** | 2.499 TL | Sınırsız | Sınırsız | Priority |

   **Neden bu fiyatlar?**
   - Basic (349 TL): Junior maaşının %0.54'ü = small business için makul
   - Pro (799 TL): Junior maaşının %1.23'ü = mid-size şirket için
   - Enterprise (2.499 TL): Junior maaşının %3.87'si = büyük şirket için

2. **Sponsorlu Yarışma** ✅
   - (Yukarıda belirtildi)
   - 50.000 TL ödül = 3.500 TL platform fee

3. **Headhunting Service** (ileride) ✅
   - Şirket: "Bu kategoride ilk 15 sanatçıyı bul"
   - Platform: Curation + outreach
   - **Ücret: 3.500 TL - 12.000 TL** (proje bazlı)

#### Örnek Flow

```
Teknoloji şirketi: "Ürün designer arıyoruz"
→ Pro Recruiter Paketi @ 799 TL/ay
→ 500+ profil view hakkı
→ Featured iş ilanı
→ Tasarımcı bulup işe aldı (direkt anlaşma)
→ Platform 0% komisyon
```

### 5️⃣ **BAĞIŞÇI / SPONSOR** (Community Support)

#### Davranış

#### Para Kazanma Kanalları

2. **Kurumsal Sponsor Paketi** ✅

| Paket | Aylık Ücret | Logo | Acknowledgment | Report |
|-------|-------------|------|----------------|--------|
| **Silver** | 2.500 TL | Evet | Site footer | Hayır |
| **Gold** | 7.500 TL | Evet | Featured | Aylık |
| **Platinum** | 15.000 TL | Evet | Homepage | Detaylı |

3. **Sustainability Membership** (ileride) ✅
   - Aylık otomatik bağış: 49 TL / 149 TL / 349 TL
   - "Bu platformu destekleyen insanlar" özel sayfası
